# operacoes_mercado_acoes
📌 Sobre o projeto

Este projeto consiste na análise de uma planilha com dados de operações no mercado de ações, com o objetivo de entender o desempenho financeiro e identificar padrões nas transações.

🎯 Objetivo
📈 Analisar lucros e prejuízos das operações
🔍 Identificar padrões de compra e venda
💰 Avaliar o desempenho geral da carteira
🛠️ Ferramentas utilizadas
📊 Excel / Google Sheets
🐍 Python (opcional)
📂 Estrutura dos dados

A planilha contém informações como:

🏷️ Ativo (ação negociada)
🔄 Tipo de operação (compra/venda)
📅 Data
🔢 Quantidade
💵 Preço
📊 Resultado (lucro/prejuízo)
📊 Análises realizadas
💰 Cálculo de lucro/prejuízo total
📉 Comparação entre operações
🏆 Identificação de ativos mais lucrativos
⏳ Análise de desempenho ao longo do tempo
📈 Resultados
✔️ Identificação das operações mais rentáveis
✔️ Visualização clara dos resultados financeiros
✔️ Apoio na tomada de decisão
▶️ Como utilizar
📂 Abra a planilha no Excel ou Google Sheets
🔎 Explore os dados utilizando filtros
📊 Crie gráficos para melhor visualização
📌 Observações

📚 Projeto desenvolvido para fins de estudo e prática em análise de dados.
